# Basic syntax

## Bold

**Bold** can be obtained with two \*.

=== "Exemple"

    ```markdown
    this is just a normal line
    **this is just a bold line**
    only this **word** is bold
    ```

=== ":right_arrow: Result"

    this is just a normal line

    **this is just a bold line**

    only this **word** is bold

## Italic

_Italic_ can be obtained with one \*.

=== "Exemple"

    ```markdown
    this is just a normal line
    *this is just an italic line*
    only this **word** is italic
    we can combine both ***bold and italic***
    ```

=== ":right_arrow: Result"

    this is just a normal line

    *this is just an italic line*

    only this **word** is italic

    we can combine both ***bold and italic***

## Strike Throught

~~Strike Throught~~ can be obtained with ~~.

=== "Exemple"

    ```markdown
    ~~this line is Strikek Throught~~
    ```

=== ":right_arrow: Result"

    ~~this line is Strikek Throught~~

## Highlighting

==Highlighting== can be obtained with ==.

=== "Exemple"

    ```markdown
    this is just a normal line
    ==this is just a Highlighted line==
    only this ==word== is Highlighted
    ```

=== ":right_arrow: Result"

    this is just a normal line

    ==this is just a Highlighted line==

    only this ==word== is Highlighted

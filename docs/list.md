# Lists

## Unordered list

=== "Code"

    ```markdown
    * alpha
    * bravo
    * charlie
        * delta
        * echo
    * foxtrot
    ```

=== ":right_arrow: Result"

    * alpha

    * bravo

    * charlie

        * delta

        * echo

    * foxtrot

## Ordered list

=== "Code"

    ```markdown
    1. alpha
    2. bravo
    3. charlie
        a. delta
        b. echo
    4. foxtrot
    ```

=== ":right_arrow: Result"

    1. alpha

    2. bravo

    3. charlie

        a. delta

        b. echo

    4. foxtrot
